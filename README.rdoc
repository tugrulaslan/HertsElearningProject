This is my University project that I have coded for 6COM1006 E-learning Applications Design And Development module at the university of Hertfordshire The application runs using MVC pattern, simply there are three roles students, visitors and lecturers -Visitors are able to visit the website and sign up in order for using the application -Students can enroll in classes, search throughout the application and see their status in dashboard, display their data as well as alter their information and password -Lecturers are enabled to add/delete courses, alter their information and password, see visitor, student and lecturer information class enrollments search keywords and all regarding statistics

Implemented Technologies Entity Framework 6.1.1 ASP.NET MVC 5.2.0 ASP.NET C#.NET .NET Framework 4.5 Bootstrap JQuery 1.9.1 Microsoft Online Team Foundation Server
